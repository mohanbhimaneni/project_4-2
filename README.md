## Final project - Batch-2

Follow these instructions to run the project:
- Install all the dependencies in the requirements.txt file in /Source/backend
-- Navigate to the said directory, then run
  ```bash
  python -m pip install -r requirements.txt
  ```
- Install all node modules in /Source/frontend
  ```bash
  npm install
  ```
- Run the backend, and the frontend
  ```bash
  python app.py
  ```
  ```bash
  npm run dev
  ```
